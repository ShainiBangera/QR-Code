# QR Code Generator using Python

## Project Overview
This project generates QR codes from text, URLs, or custom messages using Python. The generated QR code is saved as an image file and can also be displayed directly in Google Colab or Jupyter Notebook.

---

## Features
- Generate QR codes instantly
- Save QR code as PNG image
- Accepts text, URLs, and custom input
- Beginner-friendly Python project
- Works in Google Colab and Jupyter Notebook

---

## Technologies Used
- Python
- qrcode library
- Pillow library
- Google Colab / Jupyter Notebook

---

## Conclusion
This project demonstrates the use of Python libraries for generating QR codes. It is a simple and practical beginner-friendly project suitable for portfolios and mini projects.
